    yasicalc の使い方

    はじめに（最短3ステップ）

    yasicalc は、ブックごとにシートを分けて金額を集計できる計算アプリです。まずは次の3ステップで使い始められます。

    • 1. 単位を設定: 右上の「単位」に 円 / 万円 / USD などを入力します（全表示に反映されます）。
    • 2. ブックとシートを用意: サイドバーのBooksでブックを選び、必要なら「+ Add New Sheet」でシートを追加します。
    • 3. 行を入力: 「+ 行を追加」から、項目名・種類（金額/参照）・値を設定します。

    ブック・シートの操作

    • Books: ドロップダウンで対象ブックを切り替えます。
    • ＋ / ✎ / －: ブック追加・ブック名編集・ブック削除ができます（ブックは最低1つ必要です）。
    • Sheets: 一覧クリックでシート切替、デスクトップではドラッグ＆ドロップで並び替えできます。
    • シート名編集: シート画面上部のタイトル欄を直接編集します。
    • シート複製 / 削除: タイトル右のアイコンから操作できます。

    行の入力ルール

    • ＋ / －切替: 各行の左端ボタンで、加算行と減算行を切り替えます。
    • 項目名: 家賃、売上、外注費など、用途が分かる名前を入力します。
    • 種類「金額」: 数値を直接入力します。表示は自動でカンマ区切りになります。
    • 種類「参照」: 同じブック内の別シート合計を取り込みます。右側に反映額が表示されます。
    • 行を複製 / 削除: 行末のアイコンで、似た行の増殖や不要行の整理ができます。

    合計・集計の見方

    • シート画面の合計: タイトル下に、現在シートの合計が表示されます。
    • まとめて見る: サイドバー下部「まとめて見る」で、選択中ブック内のシート別合計と総合計を確認できます。
    • 循環参照エラー: A→B→A のように参照がループすると計算できません。参照先を見直して一方向にしてください。

    保存・復元

    • 自動保存: 編集内容はブラウザ内に自動保存されます（手動保存は不要です）。
    • 書き出し(JSON): 現在のデータをJSONで保存できます。バックアップ用途におすすめです。
    • 読み込み: 書き出したJSONを取り込んで状態を復元できます。
    • 初期化: 全データを消去して初期状態に戻します。実行前に書き出し推奨です。

    スマホ利用のコツ

    • シート / ブック切替: 上部の「シート」ボタンでサイドバーを開き、BooksとSheetsを操作します。
    • サイドバーを閉じる: 画面外タップで閉じられます。
    • 金額入力: 数字キーボードで入力でき、表示は自動で整形されます。
    • 横幅が狭いとき: まとめ表示の表は横スクロールで確認できます。
    • モーダルを閉じる: 「×」またはモーダル外タップで閉じられます。
    • 安全運用: 大きな変更の前は「書き出し(JSON)」で保存しておくと安心です。
