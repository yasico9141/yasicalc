    yasicalc の使い方

    はじめに（最短3ステップ）

    yasicalc は、ブックの中にシートを作って管理できる計算アプリです。必要に応じて他シートを参照しながら合計を出せます。まずは次の3ステップで使い始められます。

    • 1. 単位を設定: 右上の「単位」に 円 / 万円 / USD などを入力します。
    • 2. ブック / シートを作成: 左側のBooksでブックを切り替え、必要なら「+ Add New Sheet」でシートを追加します。
    • 3. 行を追加: 「+ 行を追加」で項目名・種類・金額（または参照先）を入力します。

    シート編集（行ごとの役割）

    • ＋ / －: 各行を加算するか減算するかを切り替えます。支出は「－」にすると管理しやすくなります。
    • 項目名: 例）家賃、食費、外注費など。あとで見返して意味が分かる名前がおすすめです。
    • 種類「金額」: 数値を直接入力する通常行です。
    • 種類「参照」: 別シートの合計を取り込む行です。カテゴリ集計や上位集計に向いています。
    • 行を複製 / 削除: 似た行をすばやく増やす、不要行を整理するのに使います。

    合計・集計の見方

    • シート画面: タイトル付近の「合計」に、そのシート単体の結果が表示されます。
    • まとめて見る: 選択中ブック内のシート合計と総合計を一覧で確認できます。
    • 循環参照エラー: A→B→A のように参照がループすると計算できません。参照関係を一方向に直してください。

    保存・復元

    • 書き出し(JSON): 現在のデータをファイル保存します。定期バックアップにおすすめです。
    • 読み込み: 書き出したJSONを取り込んで状態を復元できます。
    • 初期化: すべてのデータを削除して初期状態に戻します。実行前に書き出し推奨です。

    スマホ利用のコツ

    • シート / ブック切替: 上部の「シート」からサイドバーを開き、BooksとSheetsを切り替えます。
    • 金額入力: 数字キーボードで入力できます。カンマ付き表示にも対応しています。
    • 横幅が狭いとき: まとめ表示の表は横スクロールで確認できます。
    • モーダルを閉じる: 「×」またはモーダル外タップで閉じられます。
    • 安全運用: 大きな変更の前は「書き出し(JSON)」で保存しておくと安心です。
